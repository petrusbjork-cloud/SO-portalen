📖 Begrepp

Klicka för att vända

Lär dig begreppen — klicka på ett kort för att se förklaringen

Inuiter
Klicka för att vända →
Arktiskt ursprungsfolk vars namn betyder "Människor" på deras eget språk. Anlände till Grönland ca år 1200 från Kanada och lever där än idag.
Kolonialism
Klicka för att vända →
När ett land tar kontroll över ett annat folk och dess land, oftast med makt eller hot om makt. Danmark koloniserade Grönland från 1721.
Hjemmestyre
Klicka för att vända →
Hemstyre — det delvis egna styre som Grönland fick 1 maj 1979. Grönland fick ett eget parlament (Inatsisartut) och regering. Grönlands nationaldag.
GIUK-gapet
Klicka för att vända →
Det strategiska farvattensgapet mellan Grönland, Island och Storbritannien. Avgörande under kalla kriget — Ryssland måste passera här med sina ubåtar för att nå Atlanten.
Sällsynta jordartsmetaller
Klicka för att vända →
17 grundämnen (neodym, dysprosium m.fl.) avgörande för elbilar, vindkraftverk och militär teknik. Kina kontrollerar ~60% av världens produktion. Grönland har enorma fyndigheter.
Blocktillskott
Klicka för att vända →
Danmarks årliga bidrag till Grönland på ca 5,5 miljarder danska kronor — ungefär 40% av Grönlands hela statsbudget. Det ekonomiska beroendet som gör full självständighet så svår.
Selvstyre
Klicka för att vända →
Självstyre — Grönlands utökade självbestämmande från 2009. Grönländskan blev officiellt språk, folket erkändes juridiskt och naturresurserna tillhör nu Grönland.
Realpolitik
Klicka för att vända →
Politiskt tänkande baserat på makt och intressen snarare än regler och principer. Trumps inställning till Grönland brukar beskrivas som ett exempel på realpolitik.
Kontinentalsockel
Klicka för att vända →
Undervattensterräng som är en naturlig förlängning av landmassan. Länder kan göra anspråk på sin kontinentalsockel via FN:s havsrättskonvention — Ryssland gör stora arktiska anspråk.
Kalaallit Nunaat
Klicka för att vända →
Grönlands namn på grönländska: "Vårt land". Inte "det gröna landet" som Erik Röde kallade det — utan ett namn som rymmer stolthet, historia och rätten till självbestämmande.
Permafrost
Klicka för att vända →
Den underliggande jord som åretrunt hållit sig frusen i Arktis. När permafrosten tinar på grund av klimatförändringar sjunker hus, vägar spricker och kuster erodera.
Dekolonisering
Klicka för att vända →
Processen när koloniserade folk återfår sin självständighet och rätten att bestämma över sin egen framtid. Grönlands resa mot selvstyre är ett exempel på pågående dekolonisering.
